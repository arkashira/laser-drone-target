# Breakeven Analysis

## Cost per Active User
- **Compute**: $0.10/user/month (assuming 100ms of compute time per day at $0.0001 per 1000ms)
- **Storage**: $0.05/user/month (100MB storage per user at $0.05 per GB/month)
- **Bandwidth**: $0.05/user/month (100MB transfer per user at $0.05 per GB)
- **Total Cost per Active User**: $0.20/user/month

## Pricing Tiers
| Tier       | Price/Month | Features                                                                 |
|------------|-------------|---------------------------------------------------------------------------|
| Basic      | $20         | Basic targeting, 1 drone, 1 user, standard support                       |
| Professional| $100        | Advanced targeting, 5 drones, 5 users, priority support, analytics       |
| Enterprise | $500        | Custom targeting, unlimited drones, unlimited users, 24/7 support, API access |

## CAC Range
- **Low End**: $50 (basic marketing and onboarding)
- **High End**: $200 (enterprise sales and customization)
- **Average CAC**: $125

## LTV Estimate
- **Average Customer Lifespan**: 12 months
- **Average Revenue per User (ARPU)**:
  - Basic: $20/month
  - Professional: $100/month
  - Enterprise: $500/month
- **Average LTV**: $300 (weighted average based on expected distribution of tiers)

## Break-even Users Count
- **Total Fixed Costs**: $2,000 (initial setup, marketing, etc.)
- **Variable Cost per User**: $0.20/user/month
- **Break-even Users**:
  - Basic Tier: 100 users ($20/month × 100 users = $2,000/month)
  - Professional Tier: 20 users ($100/month × 20 users = $2,000/month)
  - Enterprise Tier: 4 users ($500/month × 4 users = $2,000/month)

## Path to $10K MRR
- **Basic Tier**: 500 users ($20/month × 500 users = $10,000/month)
- **Professional Tier**: 100 users ($100/month × 100 users = $10,000/month)
- **Enterprise Tier**: 20 users ($500/month × 20 users = $10,000/month)
